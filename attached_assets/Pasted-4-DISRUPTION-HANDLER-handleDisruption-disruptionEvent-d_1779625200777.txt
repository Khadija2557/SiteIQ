6. ROUTE PLANNER — planSafeRoute(startX, startY, endX, endY, hazards, machines):
Implement A* pathfinding on a grid:
- Grid size: 100x100 units representing the site
- Blocked cells: any cell within radius 5 of an active hazard, within radius 8 of an operating crane
- Find shortest safe path from start to end
- Return array of waypoints: [{x, y}] that the worker/vehicle should follow
- If no safe path exists, return null and trigger supervisor alert
